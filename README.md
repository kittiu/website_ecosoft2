# Ecosmart Water Solutions – Astro Site

Modern marketing website built with Astro 5, inspired by the layout and structure of ecosoft.co.th but with original design and copy.

## Getting started

```bash
cd astro-site
npm install
npm run dev
```

Then open `http://localhost:4321` in your browser.

## Scripts

- `npm run dev` – start the Astro dev server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally
- `npm run check` – Astro type and config checks
